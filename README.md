*Stack and Features*

- Tailwind/shadcnUI/radix-ui (UI and Styling)

- Zod (Model Validation)
- Clerk (Authentication, User Session, Multiple Org)
- Dnd (Drag n Drop)
- unsplash-js (Unsplash integration, fetch stock images)
- sonner (Toast, unobtrusive notifications)
- react-query
